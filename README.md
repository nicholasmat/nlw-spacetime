## 🖥️ Projeto
Esse é um projeto Web Responsivo de uma cápsula do tempo para exibir mémorias em uma linha do tempo.

##  🚀 Tecnologias
Esse prejeto foi desenvolvido duranta o NLW da Rocketseat com as seguintes tecnologias:

-HTML
-CSS
-Git e Github
